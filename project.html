REALTIME PROJECT

1. Online Feedback Mechanism for Staffs

Description :

Students from each class login with their credentials and they will be allowed to provide feedback (anonymously) for their respective staffs.The feedback form includes 20 Course outcomes and students select values in the range -2 to 5.Based on the inputs provided by the student,staffs are evaluated and the salary is provided accordingly.In admin Login,Several reports are created. Some of the reports are :

  • Individual performance of the staffs
  • Classwise Report
  • Departmentwise Report
  • Classwise Submission details

GitHub Link : https://github.com/Vignesh-24/Feedback

  • Technology Used
  • HTML,CSS,JS
  • BOOTSTRAP
  • PHP
  • AJAX
  • MySQL
